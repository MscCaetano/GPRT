Como Criar uma Página na Internet

Um guia completo e passo a passo para começar a criar suas próprias páginas web.

Passo 1: Criar o Documento HTML

O HTML (HyperText Markup Language) é a estrutura básica da sua página web. Ele define o conteúdo e a estrutura de sua página. Aqui estão os principais elementos que você usará:

  • <html> - Elemento raiz que envolve toda a página.
  • <head> - Contém meta-informações e links para estilos e scripts.
  • <body> - Contém o conteúdo visível da página.
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <h1>Bem-vindo à minha página!</h1>
    <p>Este é o conteúdo da página.</p>
</body>
</html>

Passo 2: Adicionar Estilo com CSS

O CSS (Cascading Style Sheets) é usado para estilizar sua página web. Com o CSS, você pode definir cores, fontes, layouts e muito mais. Você pode incluir CSS diretamente no HTML, mas é recomendado usar um arquivo separado.

body {
    font-family: Arial, sans-serif;
    color: #333;
}

h1 {
    color: #6e45e2;
}

p {
    font-size: 1.2em;
}

Para vincular seu arquivo CSS ao HTML, adicione a seguinte linha dentro da tag <head> do seu documento HTML:

<link rel="stylesheet" href="style.css">

Passo 3: Adicionar Interatividade com JavaScript

O JavaScript é uma linguagem de programação que permite adicionar interatividade às suas páginas web. Você pode usar JavaScript para responder a eventos, manipular o DOM e muito mais.

document.addEventListener('DOMContentLoaded', () => {
    document.getElementById('myButton').addEventListener('click', () => {
        alert('Você clicou no botão!');
    });
});

Para adicionar seu script JavaScript ao HTML, inclua a seguinte linha antes do fechamento da tag </body>:

<script src="main.js"></script>
